3.9. Mailing

Mailing is component where you can spam groups of your customers if you have to communicate something to them. Obviously you need to specify sender name and address, write your message in text box and select appropriate group of recipients. Your attendance can be grouped by status, network or user groups.

Your server has to support mail() function for this to work.

It's best to use lms-notify Perl script or notify and ggnotify modules of almsd daemon to send emails automatically whenever customer is in debt.
